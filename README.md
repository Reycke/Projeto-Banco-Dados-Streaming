# Sistema Web de Gerenciamento de Assinaturas de Streaming

## Objetivo

Desenvolver um sistema web para gerenciamento e consulta de assinaturas de streaming digital.

## Escopo

O sistema permitirá:

- Cadastro de clientes
- Cadastro de planos
- Cadastro de assinaturas
- Controle de status
- Registro de acessos
- Histórico de alterações
- Área administrativa

O sistema possui finalidade exclusivamente administrativa e informacional.# Projeto-Banco-Dados-Streaming
